# Create Hello World

Create Hello World Node.js app

## Creating an App

### npx
```bash
npx create-hello-world my-app
```

### npm
```bash
npm install -g create-hello-world
npm init hello-world my-app
```

### Yarn
```bash
yarn create hello-world my-app
```

```
my-app
├── README.md
├── package.json
├── .gitignore
└── index.js
```
